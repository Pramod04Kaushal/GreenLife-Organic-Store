# GreenLife Organic Store Management System

## Overview
GreenLife Organic Store Management System is a desktop-based application developed using C# and the .NET Framework (Windows Forms). The system is designed to manage retail operations including product management, customer management, order processing, reporting, and dashboard analytics.

This project was developed as part of the Application Development module coursework.

---

## Technologies Used

- C# (.NET Framework)
- Windows Forms
- Microsoft SQL Server
- Object-Oriented Programming (OOP)
- LINQ

---

## Features

### Admin Features
- Secure Login
- Manage Products (Add / Update / Delete)
- Manage Customers
- Manage Orders (Update order status)
- Generate Reports (Sales, Stock, Order History)
- Dashboard (Total sales, stock levels, active orders)
- Low-stock notifications
- Discount management

### Customer Features
- Registration & Login
- Search Products (Name, Category, Price range)
- Place Orders
- Track Orders
- Profile Management
- Product Ratings & Reviews

---

## Software Architecture

The system follows a 3-tier architecture:

1. Presentation Layer (Windows Forms UI)
2. Business Logic Layer (Core classes such as Product, Order, Customer)
3. Data Access Layer (DatabaseHelper class for SQL Server interaction)

Object-Oriented principles such as encapsulation, inheritance, abstraction, and polymorphism were applied.

---

## How to Run the Project

1. Open the solution file (.sln) in Visual Studio 2015 or higher.
2. Ensure SQL Server is running.
3. Configure the connection string if required.
4. Build the solution.
5. Run the application (F5).
